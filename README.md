## 🚀 Overview

Prashikshan is a scalable frontend system built to manage training workflows,  
group coordination, and structured learning experiences with a clean, intuitive UX.

---

## ✨ Core Features

- 🔐 JWT-based authentication (login & registration)  
- 👥 Group creation and management  
- 📊 Dashboard for tracking activity and progress  
- 📱 Fully responsive, mobile-first UI  
- 🔗 Secure REST API integration using Axios  
- 🧭 Seamless navigation with React Router  
- 🎬 Smooth UI animations using Framer Motion  

---

## 🛠️ Tech Stack

- React 19 (JSX)  
- Tailwind CSS v4  
- React Router DOM  
- Axios  
- Framer Motion  
- Vite  
- Deployment: Netlify
